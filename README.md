# Getting Started

## Requirements:

```
Git: 2.51.0
Spring Boot : 3.5.6
Java : 24
Maven 3.9 +
```

## Start application using:

```bash
./mvnw spring-boot:run
```

## Testing:

[http://localhost:8080](http://localhost:8080)
